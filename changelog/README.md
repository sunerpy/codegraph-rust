# Changelog archive

Per-major-version changelog snapshots live here, one file per major series:

- `CHANGELOG-v0.x.md` — the `0.x` series
- `CHANGELOG-v1.x.md` — the `1.x` series
- …

The root [`CHANGELOG.md`](../CHANGELOG.md) is the rolling, always-current changelog
generated by [git-cliff](https://git-cliff.org) (config: [`cliff.toml`](../cliff.toml))
in the release workflow's `update-changelog` job. When a major series closes, its
section is archived here so the root file can stay focused on the active series.

These files are auto-generated and excluded from `oxfmt` (see `.oxfmtignore`); do not
hand-edit them.
